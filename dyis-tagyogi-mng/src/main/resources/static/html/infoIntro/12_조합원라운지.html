사용안내

  • 한국콘텐츠공제조합에서는 조합원들을 위해 조합 내 회의실 등을 개방 하고 있습니다.
  • 사용대상 : 한국콘텐츠공제조합 조합원에 한해 사용가능합니다.
  • 사용시간 : 평일 오전 9시부터 저녁 6시까지 입니다. ( ※ 사용시간 연장은 별도 문의 바랍니다. )
  • 사용요금 : 별도의 사용요금은 없습니다.
  • 시설 및 예약문의 : 02-3661-5383
  • 담당부서 : 기획팀

열린회의실

  • 대회의실(15석 규모) 및 소회의실
  • 프로젝터, 화이트보드 등이 갖추어져 있습니다. ( 예약 후 사용이 가능합니다. )

라운지

  • Wi-Fi 사용이 가능하며, 조합원은 조합 업무시간내 사용이 가능합니다.
