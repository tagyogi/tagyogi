보증상품 추진절차

입찰보증

  • 상품설명
  • 각종 계약의 수주를 위한 입찰 참가에 필요한 입찰보증금을 대신하여 이용하는 상품입니다.
  • 이용고객
  • 계약자 : 계약수주자(수요자)
  • 피공제자 : 계약발주자(공급자)
  • 보증금액
  • 입찰금액의 일정비율 이상
  • 보증기간
  • 개시일 : 입찰서 제출마감일
  • 종료일 : 마감일 다음날부터 30일 이후
  • 보상내용
  • 입찰참가자가 입찰에 참가하여 최종 낙찰자로 선정되었음에도 불구하고 보증기간 내에 주 계약을 체결하지 아니함으로써 피공제자가 입은 손해액을 보상

계약보증

  • 상품설명
  • 각종 계약 체결 시 계약의 완료를 목적으로 계약보증금을 대신하여 이용하는 상품입니다.
  • 이용고객
  • 계약자 : 계약수주자
  • 피공제자 : 계약발주자
  • 보증금액
  • 계약금액의 일정비율 이상
  • 보증기간
  • 개시일 : 계약기간 입찰개시일
  • 종료일 : 계약기간 종료일이후
  • 보상내용
  • 계약자가 계약상의 의무를 보증기간내에 이행하지 아니함으로써 피공제자가 입은 손해액을 보상

하자보증

  • 상품설명
  • 계약이행 완료 후 일정기간 동안 하자보수의무를 목적으로 하자보증금을 대신하여 이용하는 상품입니다.
  • 이용고객
  • 계약자 : 계약수주자
  • 피공제자 : 계약발주자
  • 보증금액
  • 계약금액의 일정비율 이상
  • 보증기간
  • 개시일 : 목적물을 인수한 날과 준공검사를 완료한 날 중 먼저 도래한 날
  • 종료일 : 하자담보기간 종료일 이후
  • 보상내용
  • 계약자가 용역 등의 사용검사 또는 검수를 받은 후 하자담보 책임기간 내에 발생한 하자에 대하여 그 보수 이행청구를 받았음에도 이를 이행하지 아니함으로써 피공제자가 입은 손해액을 보상

선급금보증

  • 상품설명
  • 각종 계약체결 후 계약 완료전에 발주자로부터 기성대가 이외에 대금을 먼저 지급받는 선금에 대한 반환 보증상품입니다.
  • 이용고객
  • 계약자 : 계약수주자
  • 피공제자 : 계약발주자
  • 보증금액
  • 선금금액
  • 보증기간
  • 개시일 : 선금수령일
  • 종료일 : 계약종료일
  • 보상내용
  • 계약자가 계약과 관련한 선금급 채무를 이행하지 아니함으로써 피공제자가 입은 손해액을 보상

이행지급보증

  • 상품설명
  • 도급계약, 매매계약, 임대차, 위임, 임차, 소비대차계약 및 기타계약에 수반하여 발생하는 채무이행보증금을 대신하여 이용하는 상품입니다.
  • 이용고객
  • 계약자 : 계약수주자
  • 피공제자 : 계약발주자
  • 보증금액
  • 채무이행보증금
  • 보증기간
  • 개시일 : 계약체결일
  • 종료일 : 이행기일의 말일
  • 보상내용
  • 계약자가 계약에서 정한 채무를 이행하지 아니함으로써 피공제자가 입은 손해액을 보상
