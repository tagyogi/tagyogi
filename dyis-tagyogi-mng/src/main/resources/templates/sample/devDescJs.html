ajax기본호출 com.fnAjaxDef json ( url, param, callBack, loadYn, sessYn, msgYn, moveUrl) 설명)
param: 메소드에게 넘길 변수, callBack : 콜백함수, loadYn :로딩이미지표시여부,
sessYn:세션여부체크, msgYn: 처리메세지출력여부, moveUrl:처리후 페이지호출url
예시)
param = {sMembNo:sMembNo, sMembNm:sMembNm };
obj = {url:"/com/selectMembList.do", param:param, callBack:com.fnMembInfoCallBack, loadYn:"Y"};
com.fnAjaxDef(obj);
ajax호출(post) com.fnAjaxPost json ( url, param, callBack, loadYn, sessYn, msgYn, formId) 설명)
param: 메소드에게 넘길 변수, callBack : 콜백함수, loadYn :로딩이미지표시여부,
sessYn:세션여부체크, msgYn: 처리메세지출력여부, formId:formId
예시)
obj = {url:"/com/selectMembList.do", formId:"saveForm", callBack:com.fnMembInfoCallBack, loadYn:"Y"};
com.fnAjaxDef(obj);
ajax호출(Multi) com.fnAjaxMulti json ( url, param, callBack, loadYn, sessYn, msgYn, formId, fileId) 설명)
param: 메소드에게 넘길 변수, callBack : 콜백함수, loadYn :로딩이미지표시여부,
sessYn:세션여부체크, msgYn: 처리메세지출력여부, formId:formId fileId:fileId공통부분
예시)

com.fnAjaxDef(obj);
